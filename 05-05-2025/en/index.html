HTML

HTML (HyperText Markup Language) is the standard language used to create and design the structure of web pages. It is the foundation of nearly all websites and plays a critical role in web development.

Key Features of HTML

  1. Markup Language:
    HTML uses a system of tags to define elements on a web page. Tags are enclosed in angle brackets (< >) and come in pairs (an opening tag and a closing tag).
  2. Structure of a Web Page:
    HTML provides the basic structure of a web page, which is typically divided into:
    • Head: Contains metadata like title, description, and links to external resources (CSS, JavaScript).
    • Body: Contains the main content of the web page.
  3. Flexibility:
    HTML supports embedding other technologies like CSS (for styling) and JavaScript (for interactivity).

Basic Structure of an HTML Document

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Basic HTML Page</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
    <p>This is a simple example of an HTML document.</p>
</body>
</html>

Explanation:

  • <!DOCTYPE html>: Declares the document type and version (HTML5).
  • <html>: Root element of the HTML document.
  • <head>: Contains metadata (not displayed on the page).
  • <body>: Contains the visible content of the page.
