HTML

HTML (HyperText Markup Language) គឺជាភាសាស្តង់ដារដែលប្រើសម្រាប់បង្កើត និងរចនារចនាសម្ព័ន្ធនៃទំព័របណ្តាញ។ វាជាមូលដ្ឋានសម្រាប់គេហទំព័រភាគច្រើន និងមានតួនាទីសំខាន់ក្នុង web development។

លក្ខណៈសំខាន់ៗនៃ HTML

  1. Markup Language:
    HTML ប្រើប្រព័ន្ធនៃស្លាកដើម្បីកំណត់ធាតុនានានៅលើទំព័របណ្តាញ។ ស្លាកត្រូវបានបិទក្នុងសញ្ញាក្រចក (< >) ហើយមានជាគូ (ស្លាកបើក និងស្លាកបិទ)។
  2. រចនាសម្ព័ន្ធនៃ Web Page :
    HTML ផ្តល់សម្ព័ន្ធមូលដ្ឋាននៃ Web Page ដែលជាធម្មតាត្រូវបានបែងចែកជា:
    • Head: មានទិន្នន័យមេតាដូចជាចំណងជើង ការពិពណ៌នា និងតំណភ្ជាប់ទៅធនធានខាងក្រៅ (CSS, JavaScript)។
    • Body: មានខ្លឹមសារសំខាន់នៃទំព័របណ្តាញ។
  3. ភាពបត់បែន:
    HTML គាំទ្រការបញ្ចូលបច្ចេកវិទ្យាផ្សេងៗដូចជា CSS (សម្រាប់រចនាស្អាត) និង JavaScript (សម្រាប់អន្តរកម្ម)។

រចនាសម្ព័ន្ធមូលដ្ឋាននៃ HTML file

<!DOCTYPE html>
<html lang="km">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ទំព័រ HTML មូលដ្ឋាន</title>
</head>
<body>
    <h1>ស្វាគមន៍មកកាន់ HTML</h1>
    <p>នេះគឺជាឧទាហរណ៍សាមញ្ញនៃឯកសារ HTML។</p>
</body>
</html>

ការពន្យល់:

  • <!DOCTYPE html>: ប្រកាសប្រភេទឯកសារ និងកំណែ (HTML5)។
  • <html>: ធាតុឫសនៃឯកសារ HTML។
  • <head>: មានទិន្នន័យមេតា (មិនបង្ហាញនៅលើទំព័រ)។
  • <body>: មានខ្លឹមសារដែលអាចមើលឃើញនៅលើទំព័រ។
